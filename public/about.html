About VoiceUp — Grievance Management System

VoiceUp is an intuitive, secure platform built to collect, track and resolve student grievances efficiently. It brings transparency to the grievance redressal workflow by providing a single platform for students, admins, and VIP staff.

Project Overview

VoiceUp allows students to submit grievances (academic, hostel, examination, library, etc.) and enables administrators to triage, update, and resolve them with status updates and remarks.

Key Features
Submit Grievance

Students can submit details and attachments; timestamped and stored.

Admin Management

Admins update status, add remarks, resolve or reject grievances.

VIP Panel

A special dashboard for priority-level grievance handling.

Feedback Module

Students provide feedback; admins can view responses.

Audit Trail

Each action is tracked for transparency.

Responsive UI

Optimized for mobile using Bootstrap 5.

Tech Stack
  • Backend: Node.js, Express.js
  • Frontend: HTML, CSS, Bootstrap
  • Database: MongoDB (Mongoose ODM)
  • Authentication: Express-session + bcrypt.js
  • Hosting: Render / Railway / VPS / Local Deployment
How It Works — Workflow
1
Student logs in

They land on the dashboard.

2
Submit Grievance

Details stored with default status.

3
Admin Reviews

Seen → In Progress → Resolved / Rejected.

4
Student Views Updates

Can view remarks, decisions & provide feedback.
